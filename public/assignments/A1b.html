Requirements

For this assignment, you are expected to implement two main components to generate a neural SDF and then render it in a GLSL environment with ray marching. We will briefly discuss each component as follows.

Step 1: Neural SDF Creation

In this step, you are asked to create a pytorch Dataset class named NeuralSDFDataset which loads a 3D mesh with .obj format and sample data points for training. After this, you will create a network to train Neural SDF. Finish the training loop and train your network and tune your network hyperparameters if you have time.

Step 1.1 Preparation

Upload our neural_sdf.ipynb to Google Colab following the tutorial given in Tutorials/Google Colab Tutorial. Run all the cells before Step 1.2. These cells will install required packages and give helper functions we shall need later to convert network weights to matrices that can be used in our shader to represent the trained network.

Step 1.2 NeuralSDFDataset

In this step, you will implement a NeuralSDFDataset class. We will follow the the way Neural SDF paper sample training points where more points are sampled near the mesh surface and less points in the space of a unit sphere. This functionality is provided in a package called mesh_to_sdf and we shall take advantage of this. You should go to the place where it says Step 1.2 in the comments and perform the following steps.
(1) You will first use a package called trimesh (it's already imported in Step 1.1) to load a .obj file with path mesh_path.

(2) After loading this mesh, you will use sample_sdf_near_surface function from mesh_to_sdf package to sample training points for this mesh. You should sample sample_num number of points for training.

(3) After sampling points, you will convert sampled points in type numpy ndarray to torch tensor by using the torch.from_numpy function. Since we are using Colab where GPU is available, you will put those tensors to CUDA by using .to(device).

You should see a point cloud like this for bunny and cow.

Step 1.3 Network Structure

After the dataset is finished. You will be tasked to create a Network class called NeuralSDF. This consists of two parts: 1. a helper class for creating identical layers called SineLayer, 2. the NeuralSDF Layer.

(1) You should fill in the constructor of SineLayer class. This requires you to first create a Linear layer with input features size in_features and output feature size out_features.

(2) You should fill in the forward function. If is_last is True, then you should not apply any activation functions (i.e. sin activation) or skip connections to this layer. If is_first is True, then you should apply the sin activation function but not the skip connection. Otherwise, you should apply both sin activation and skip connection.
For sin activation, it should be in form sin(omega * f(x)) where omega being sin activation frequency and f(x) being the output of linear layer.
For skip connection, it should be in form current_layer(x) / skip_weight + x where x being the input to current layer, current_layer being sin(omega * f(x)) and skip_weight is given in the class constructor.

(3) After finish the SineLayer class, lets move to finish the last piece of the network structure, i.e. put every together. In NeuralSDF, you should use the SineLayer class to create our network structures. Use is_first=True for the first layer and is_first=False otherwise. Use is_last=True for the last layer and is_last=False otherwise. You should create one first layer which has input feature size in_features and output feature size hidden_features. You should then create hidden_layers number of hidden layers with both input and output feature size of hidden_features. Finally, you should create the last layer with input features size hidden_features and output features size being 1. You should use the skip weights being set to skip_weight=sqrt(i+1) where i is the current index of hidden layer.

Step 1.4 Train Your Network

Finally, you will finish the training loop. You will use Mean Squared Loss (MSE) as our loss function. Remember to zero the grad in your optimizer at the start of each iteration and propagates the loss backward at the end of each iteration.

Now, train your network! It should be fast which will finish in several minutes. After you finish everything and if you have time, feel free to play with the parameters and network structures.

Step 4: Ray Marching

In this step, you are asked to implement the ray marching algorithm to render the SDF scene. You are allowed to reuse your previous implementation in A1a for this function.

Step 5: Normal Calculation

In this step, you will calculate the surface normal at a given point using the finite difference method. You are allowed to reuse your previous implementation in A1a for this function.
